Exercício Prático: Criando uma Página Web

Crie um novo repositório público em seu GitHub chamado avaliacao-frontend. Nesse diretório, crie uma página de perfil básica usando HTML e CSS. A página deve ter a seguinte estrutura: 1. Um cabeçalho contendo seu nome e uma foto de perfil. 2. Uma seção de informações pessoais, contendo seu nome, idade, cidade e ocupação. 3. Uma seção de descrição, onde você pode escrever uma breve descrição sobre si mesmo. 4. Uma seção de interesses, onde você lista alguns de seus hobbies ou interesses. 5. Um rodapé contendo seus contatos, como e-mail e links para perfis sociais.

Use as tags semânticas apropriadas para estruturar o conteúdo. Estilize a página usando CSS para torná-la visualmente agradável. Aplique estilos aos elementos, como cores de fundo, fontes, tamanhos e margens. Utilize seletores CSS para estilizar elementos específicos, como o cabeçalho, seções e rodapé. Explore propriedades de posicionamento e layout para organizar os elementos na página. Adicione pelo menos uma imagem de perfil e ajuste seu tamanho usando CSS. Utilize cores e estilos de sua preferência para personalizar a aparência da página.

Dicas: Lembre-se de manter o código HTML e CSS organizado e bem estruturado. Use classes e IDs para selecionar elementos específicos no CSS. Faça uso das propriedades de espaçamento, como `margin` e `padding`, para ajustar o espaçamento entre os elementos. Utilize propriedades de fonte, como `font-family` e `font-size`, para personalizar a tipografia. Entrega: Faça o deploy (publicação) da página utilizando o GitHub Pages (não se esqueça de que esse serviço pode levar alguns minutos para completar a publicação da página). Crie um arquivo README.md no repositório contendo o link do deplyment (site publicado).
